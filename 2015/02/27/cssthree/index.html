带三角的小气泡的实现

By hey

2015-02-27 更新日期:2015-02-28

文章目录

知乎上的传说一道百度的题目，就是实现一个带三角的小气泡

##方案一

思维比较简单的方法是用特殊字符来做，代码如下：

<!DOCTYPE HTML>
<html lang="en-US">
<head>
    <meta charset="UTF-8">
    <title>字符制作三角边框</title>
    <style type="text/css">
    *{margin: 0;padding: 0;}
    .demo1{border:3px solid #0B20B9;width: 158px;height: 158px;background-color: #87CEEA;}
    .demo1 .s1,.demo1 .s2{display: block;font-size: 50px;}
    .demo1 .s1{margin-left: 139px;color:#0B20B9;}
    .demo1 .s2{margin-left: 135px;margin-top: -67px;color:#87CEEA;}
    </style>
</head>
<body>
    <h1>字符制作三角形边框</h1>
    <div class="demo1">
        <span class="s1">&#9670;</span>
        <span class="s2">&#9670;</span>
    </div>
</body>
</html>

这种利用特殊字符的方法，不符合标签语义化，但是实现起来比较简单

##方案二：
也是添加额外标签，但是简明扼要一些，代码如下：

<!DOCTYPE HTML>
<html lang="en-US">
<head>
    <meta charset="UTF-8">
    <title>纯css制作三角边框</title>
    <style type="text/css">
    #test{
        width:158px; 
        height:158px;
        border:3px solid #0B20B9;
        background-color: #87CEEA;
        position: relative;
    }
    #test em{
        display: block;
        position: absolute;
        right: -40px;
        top:15px;
        line-height: 0;
        font-size: 0;
        border-style:   dashed dashed dashed solid;
        border-width: 20px;
        border-color:  transparent transparent transparent #0B20B9;

    }
    #test span{
        display: block;
        position: absolute;
        right: -36px;
        top:15px;
        line-height: 0;
        font-size: 0;
        border-style:   dashed dashed dashed solid;
        border-width: 20px;
        border-color:  transparent transparent transparent #87CEEA;

    } 
    </style>
</head>
<body>
    <h1>纯css制作三角边框</h1>
    <div id="test">
        <em></em>
        <span></span>
    </div>
</body>
</html>

line-height与font-size都设置为0，可以搞定ie6边框左右梯形的问题，让边框呈现三角形。将border-style设置为dashed解决ie6 transparent不能用默认为黑色的情况。

##方案三：
不添加em与span，利用before与after代码如下：

<!DOCTYPE HTML>
<html lang="en-US">
<head>
    <meta charset="UTF-8">
    <title>css before与after制作三角边框</title>
    <style type="text/css">
    .tag{
        width:300px;
        height:100px;
        border:5px solid #09F;
        position:relative;
        background-color:#FFF;
    }
    .tag:before,.tag:after{
        content:"";display:block;
        border-width:20px;
        position:absolute; bottom:-40px;
        left:100px;
        border-style:solid dashed dashed;
        border-color:#09F transparent transparent;
        font-size:0;
        line-height:0;
    }
    .tag:after{
        bottom:-33px;
        border-color:#FFF transparent transparent;
    }
    </style>
</head>
<body>
    <h1>css before与after制作三角边框</h1>
    <div class="tag">
    </div>
</body>
</html>
