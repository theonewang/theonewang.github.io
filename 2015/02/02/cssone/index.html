css(一)清除浮动与常见居中

By hey

2015-02-02 更新日期:2015-02-02

文章目录

##常用清除浮动：

.clear:after{
              content: ".";
              height: 0;
              visibility: hidden;
              display: block;
              clear: both;
            }

##每一页主内容居中的方法：

.mainpart{
            width: 966px;
            position: relative;
            left: 50%;
            margin-left: -483px;
        }

使用到相对定位,然后定位50%的位置，再使左外边距为负数
